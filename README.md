# Chariot

A macOS desktop app for managing AI agent coding loops.

Chariot harnesses the power of large language models and drives autonomous AI coding loops — you define a goal, and the agent iterates, reasons, and executes until the task is complete.

## Download

Go to [Latest Release](https://github.com/xinghui/chariot-releases/releases/latest) to download the latest `.dmg`.

## Install

1. Download `Chariot_<version>_aarch64.dmg`
2. Open the `.dmg`
3. Drag **Chariot** to your **Applications** folder
4. Launch from Applications

> On first launch, right-click the app and select **Open** to bypass Gatekeeper.
